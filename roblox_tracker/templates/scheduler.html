How it works: Every user you look up via the search page is automatically added to the tracking list. The scanner re-scans all tracked users every {{ (scanner.interval // 3600) }} hour(s), saving fresh snapshots of their games, friends, and groups. This lets the Changes and Past Groups tabs detect activity even when you're not actively checking.

Condo Discovery: Every 6 scan cycles, the scanner automatically cross-references members of known condo groups to discover new ones. You can also run discovery manually.

Persistence: The tracked-users list, all snapshots, and discovered condo groups are saved to disk. When the server restarts, scanning resumes automatically. Scans only run while the server process is running — if you stop the server, no scans occur until you start it again.
